Privacy Policy

Effective Date: April 24, 2025

1. Introduction

This Privacy Policy explains how Dustin Decker (“we,” “our,” or “us”) handles information when you use the Malicious Link Scanner proof-of-concept application (“App”). By installing or using the App, you agree to these practices.

2. Information We Collect

  • Message Data: Text of incoming messages from Meta platforms (Messenger, Instagram Direct, WhatsApp Business).
  • Extracted URLs: Any web links parsed from those messages.
  • Meta Identifiers: Sender or thread IDs supplied by Meta APIs (no additional personal details).
  • Usage Metrics: Timestamps and scan-count stats for performance monitoring.

3. How We Use Your Information

  • Threat Analysis: Forward each URL to the VirusTotal Premium API for security scanning.
  • App Functionality: Enable message parsing, rate-limit handling, and error logging.
  • Performance Monitoring: Collect anonymized metrics to ensure smooth operation.

4. Data Retention

We do not store message text, URLs, or scan results beyond the moment they are processed. All data is discarded immediately after each scan.

5. Information Sharing

  • VirusTotal: URLs you submit go directly to VirusTotal under their own policy.
  • Meta Platforms: No data is shared back to Facebook/Instagram/WhatsApp beyond normal API calls.
  • No Third-Party Distribution: We do not sell, rent, or distribute your data.

6. Security

We use HTTPS for all webhooks and API calls and secure storage for keys. No system is 100% secure, but we follow best practices.

7. Children’s Privacy

This App is not for users under 13; we do not knowingly collect data from anyone under 13.

8. Changes to This Policy

We may update this policy at any time; the “Effective Date” will reflect the latest revision. Continued use of the App means you accept any changes.
